1. Що таке керований (managed) код?
Відповідь: Керований код — це код, виконання якого контролюється середовищем CLR (Common Language Runtime). CLR відповідає за управління пам'яттю (збирання сміття), безпеку типів, обробку винятків тощо.

2. Що таке CLR?
Відповідь: CLR (Common Language Runtime) — це середовище виконання .NET, яке забезпечує виконання керованого коду. Воно надає такі послуги, як JIT-компіляція, управління пам'яттю (Garbage Collection), безпека типів та обробка винятків.

3. String — це value type чи reference type?
Відповідь: String (рядок) — це reference type (посилальний тип). Хоча він поводиться як value type у деяких випадках (наприклад, незмінність/immutability), він зберігається в купі (heap), а змінна містить посилання на об'єкт.

4. Що таке інкапсуляція?
Відповідь: Інкапсуляція — це принцип ООП, який полягає у приховуванні внутрішньої реалізації об'єкта та наданні доступу до даних лише через публічні методи або властивості. Це захищає дані від некоректного використання.

5. Що таке поліморфізм?
Відповідь: Поліморфізм — це здатність об'єктів різних типів оброблятися як об'єкти одного базового типу. Це дозволяє викликати один і той самий метод для різних об'єктів, і кожен об'єкт буде виконувати його по-своєму (наприклад, через перевизначення методів).

6. Що таке абстрактний клас?
Відповідь: Абстрактний клас — це клас, який не може бути інстанційований (не можна створити його екземпляр) і може містити абстрактні методи (без реалізації), які повинні бути реалізовані в класах-спадкоємцях.

7. Що таке інтерфейс?
Відповідь: Інтерфейс — це контракт, який визначає набір методів і властивостей, які повинен реалізувати клас. Інтерфейс не містить реалізації методів (до C# 8.0) і полів, лише сигнатури. Клас може реалізовувати багато інтерфейсів.

8. Що таке конструктор?
Відповідь: Конструктор — це спеціальний метод класу, який викликається автоматично при створенні екземпляра об'єкта. Він використовується для ініціалізації полів об'єкта та виконання початкових налаштувань.

9. Що таке ref параметри методів?
Відповідь: Ключове слово ref дозволяє передавати аргумент у метод за посиланням, а не за значенням. Це означає, що будь-які зміни параметра всередині методу вплинуть на оригінальну змінну, передану ззовні. Змінна повинна бути ініціалізована перед передачею.

10. Що таке властивість?
Відповідь: Властивість (Property) — це член класу, який надає гнучкий механізм для читання, запису або обчислення значення приватного поля. Властивості виглядають як публічні поля, але насправді є спеціальними методами (аксесорами get і set).
